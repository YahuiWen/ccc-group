# Backend 

Koa2 and web socket

## Backend Setup
### clone the project
git clone https://github.com/YahuiWen/ccc-group

### enter the project directory
cd backend
### install dependency
npm install

### run
node app.js



# Frontend Setup

vue, element-ui, echarts

## enter the project directory
cd frontend
## install dependency
npm install

## run
npm run serve
This will automatically open http://localhost:8080

# Build
## run build to genertate dist file.
npm run build
